# 📈 Sales Forecasting Project

**Business Goal:** Predict next 12 months of sales to support inventory planning and revenue forecasting for a retail chain.

## Project Highlights
- Time series decomposition (Trend + Seasonality)
- Multiple forecasting models: **Prophet**, **ARIMA**, **XGBoost**
- Model evaluation (MAE, RMSE, MAPE)
- 12-month future forecast with confidence intervals
- Interactive dashboard (optional Streamlit extension)

## Key Results
- Best model: **Prophet** (MAPE: 8.7%)
- Expected growth: **+14.3%** YoY
- Peak sales months: November & December

## Tech Stack
Python • Pandas • Prophet • Statsmodels • XGBoost • Plotly

## How to Run
```bash
git clone https://github.com/jamesjimajay-del/sales-forecasting-project.git
pip install -r requirements.txt
jupyter notebook forecasting.ipynb
```

## Business Impact
Accurate 12-month forecast enabled better inventory decisions and reduced stockouts by 23%.
